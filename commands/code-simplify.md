---
command: /code-simplify
skill: code-simplification
---

Identify and apply simplification opportunities — one change at a time, verify after each.

**Usage:**
- `/code-simplify` — Simplify active feature
- `/code-simplify <file-or-dir>` — Simplify specific target

**Pattern:** Propose → wait → execute → test → report (one at a time)

**Output:** Simplified code with improved readability
